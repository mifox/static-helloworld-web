# Static Hello World Web Page

这是一个简单的静态HTML页面示例，展示了基本的前端技术。

## 项目结构

- `index.html` - 主页面文件
- `style.css` - 样式表文件
- `script.js` - JavaScript交互文件
- `README.md` - 项目说明文件

## 功能特性

1. 响应式设计，适配不同屏幕尺寸
2. 美观的渐变背景和卡片式布局
3. 简单的JavaScript交互功能
4. 点击页面可触发动态效果
5. 标题颜色定时变化

## 如何使用

1. 直接在浏览器中打开 `index.html` 文件即可查看页面
2. 或者将所有文件部署到Web服务器上访问

## 浏览器兼容性

- Chrome (推荐)
- Firefox
- Safari
- Edge
